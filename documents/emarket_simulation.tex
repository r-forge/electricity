\author{Anton Burger and Robert Ferstl}
\maketitle
\abstract{
The paper discusses a model for capacity investment decisions in a deregulated electricity market. The problem is formulated as dynamic oligopoly game and solved under different information structures, i.e. open-loop, closed-loop, feedback. An example is given for the Austrian wholesale electricity market.
%We solve an asymmetric n-player oligopoly problem with a simulation approach. Each company can choose between several production technologies, which differ in terms of fixed and variable costs. We use empirical data of the Austrian electricity market for the parameters of the cost functions. The results allow quantifying the impact of regulatory measures, e.g. number of players, strategic 
%behaviour, market design and technology choice.
}
